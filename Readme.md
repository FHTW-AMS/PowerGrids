# Python materials for AMS - Group "Power Grids"

This repository collects some notebooks and Python files.

They cover different areas of data acquisition and modeling of energy systems as well as power grids.

The notebooks contain explanations and best practices and will be expanded during the semester as needed.

The repository can be downloaded after installing git with the command shown above.

```bash
git clone https://github.com/FHTW-AMS/PowerGrids.git
```

The notebook *00_prerequisites.ipynb* describes the use of version control,
the installation of Python, programming environments, and the general setup.

The other notebooks contain Python code and explanations.

## Course overview

Planned content, simulation of 
- electric grids,
- generation, or 
- consumption 

with Python.

* Setup
    * Version control
    * Python installation and packages
    * Programming interface

* Developing individual components, refreshing Python basics
    * Standard load profiles: Working with time series and functions
    * Battery: Building and Using classes
    * PV: Using an external library 

* Power grid simulation with pandapower
    * Building a network from basic components
    * Power flow calculation and results
    * Using example networks
    * Combining own profiles and the temporal evolution of the power flow
Other topics include PowerFactory (not covered in this repository), followed by project selection and execution in small groups.

Depending on demand, we will also cover further topics together, which will then be added here as well.
